# Sistem Pendukung Keputusan Penilaian Kharismatik Guru Metode Topsis

Sistem Pendukung Keputusan Penilaian Kharismatik Guru Metode Topsis merupakan sistem penilaian kharismatik pada guru, penilaian diberikan oleh siswa dan dengan adanya sietem ini dapat memberikan evaluasi kegiatan belajar mengajar
